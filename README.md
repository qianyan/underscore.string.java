===

A library for Java String manipulation
## Prerequisites
* java >=1.6
* guava 18.0

## Installation
### gradle
```groovy
repositories {
    maven {
        url 'https://oss.sonatype.org/content/groups/public'
    }
}

dependencies {
    compile ("com.lambeta:underscore.string.java:0.0.1-SNAPSHOT")
}
```
